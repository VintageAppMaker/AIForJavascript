### ChatGPT

아래는 주어진 pseudocode를 JavaScript로 변환한 소스 코드입니다:

```javascript
function max(num1, num2) {
  if (num1 > num2) {
    return num1;
  } else {
    return num2;
  }
}

// 시작
var 결과 = max(3, 5); // max 함수에 3과 5를 입력
console.log(결과); // 결과값 출력

```

위의 소스 코드를 실행하면 결과로 5가 출력될 것입니다.
